# Organization API

REST API приложение для справочника организаций, зданий и видов деятельности.

## Технологии

- FastAPI
- SQLAlchemy
- Alembic
- PostgreSQL
- Docker

## Установка и запуск

### Через Docker (рекомендуется)

1. Клонируйте репозиторий
2. Запустите контейнеры:

```bash
docker compose up --build
```
